Mar 20, 2016 | Hits

突然间想做一个Blog, 记录一些曾经该记录的东西。20多岁时该做而 又没做的事，如今来做。只能说出来混的，迟早是要还的。Google了一下，意外地发现一种有逼格的写博客的方式 。来本文记录了本Blog（Hexo + Markdown + Github）搭建的完整过程。

##相关工具

Github：免费的静态站点，三个特点：免费托管、自带主题、支持自制页面。

Hexo：一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

Markdown：能让文档更容易读、写和随意改。
