搭建完一个blog后，还需要进行一些简单的定制。比如说分类、标签、关于等等。本文记录了我的定制。

换个主题

已经在前一篇中插述。

配置 分类

手动添加

当 hexo n name 后，默认的 .md 只有以下三项，没有分类的。

我们只需要在 tags: 下一行加上 “ categories: 工具 ” 即可。

配置模板

每个博客都手动添加 categories: 比较麻烦，我们可以添加一个模板，模板位于scaffolds目录下，执行新建命令时，是根据这里的模板页来完成的，于是可以在这里根据需求定义一些默认值。

打开 scaffold/post.md 在 tags: 下一行加上 categories:.

配置 标签

新建的博客中已经默认有 tags:, 只需要将想要的标签添加到后面即可，对于添加多个标签，需要将标签放在 [ ] 之中，并且用 , 隔开。示例：

标签 也可以像 分类 那样配置路径，编辑_config.yml文件，找到 tag_map: ，其它同 分类 。

配置显示 目录

给博客显示个 目录 很简单，只需要在 tags: 下一行加上 “ toc: true ” 即可。可以将此处配置到模板中一劳永逸。加上此配置后，hexo会跟Markdown的标题自动生成 目录。
