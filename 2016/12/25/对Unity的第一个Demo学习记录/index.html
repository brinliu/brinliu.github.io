创建游戏面板(游戏区域)

通过Unity Plane来实现。操作：

void OnTriggerEnter (Collider other)
{
if (other.gameObject.CompareTag (“Pick Up”)) {
other.gameObject.SetActive (false);
}
}
```

####设置Tag

在脚本中判断了Game Object的Tag, 因此需要将Pick Up设置为”Pick Up”。

在Prefabs中选中Pick Up,Inspector - Tag - Add Tag,进入Tag manager. 在Tag List中添加一个Tag 0名为”Pick Up”。保存后，在Inspector - Tag选择刚才添加的Tag名字。

####优化

static colliders shouldn’t move. Dynamic colliders can move, and have a rigid body attached.Standard rigid bodies are moved using physics forces. Kinematic rigid bodies are moved using their transform.
